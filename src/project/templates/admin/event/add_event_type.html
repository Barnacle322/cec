Что такое "Тип события"?

Тип события — это категория, которая помогает пользователям фильтровать и находить интересующие их мероприятия.

Рекомендации по названию:

  • Используйте короткие и понятные названия (1-2 слова).
  • Примеры: Воркшоп, Лекция, Хакатон, Встреча.
  • Избегайте дублирования (например, не создавайте "Мастер-класс" если уже есть "Воркшоп").
